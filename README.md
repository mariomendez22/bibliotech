Sistema BiblioTech

Sistema bibliotecario desarrollado con Flutter, Node.js y MySQL para la gestión de préstamos bibliográficos.

Tecnologías
- Flutter
- Dart
- Node.js
- Express
- MySQL

Funcionalidades
- Inicio de sesión
- Registro de préstamos
- Registro de devoluciones
- Consulta de préstamos activos
- Consulta de historial
- Consulta de libros
- Consulta de socios
- Control de disponibilidad de ejemplares

Node:
npm install
npm start

Flutter:

flutter pub get
flutter run
